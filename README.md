# KV Store - Demo application

## Logic (Rust)

```bash
pnpm run logic:build
```

Optional cleanup:

```bash
pnpm run logic:clean
```

## App (React)

```bash
cd app && pnpm install
```

Build and run:

```bash
pnpm --dir app build
pnpm --dir app dev
```

Open the app in your browser and connect to a running node.

Docs: https://calimero-network.github.io/build/quickstart
